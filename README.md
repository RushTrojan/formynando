## 💜 Support the Project

### 🌟 How ARMY Can Contribute

We need YOUR help to make this the ultimate BTS discography! Here's how you can contribute:

#### 1️⃣ **Fix Broken Video Links**
Some YouTube video IDs may be incorrect or broken. You can help!
- Click the **🔧 Fix Videos** button
- Search for the correct video on YouTube
- Copy the video ID and update it
- Submit your changes!

#### 2️⃣ **Add Missing Songs**
Know a song that's missing? 
- Fork this repository
- Add the song details to the `allSongs` array
- Submit a pull request
- Include: title, artist, album, year, type, youtubeId, image

#### 3️⃣ **Improve Translations**
- Add Korean titles (한글)
- Add romanized lyrics
- Improve album descriptions

#### 4️⃣ **Report Issues**
Found a bug? Song info incorrect? 
- Open an [Issue](https://github.com/shafaq0410/BTS_Discography/issues)
- Be specific about the problem
- We'll fix it ASAP!

#### 5️⃣ **Spread the Purple Love** 💜
- ⭐ Star this repository
- 🔗 Share with fellow ARMY
- 💬 Tell us your favorite features
- 📱 Share on social media with #BTSDiscography

## 📖 How to Use

### For Listeners 🎧

1. **Browse** - Scroll through the sidebar to see all songs
2. **Search** - Use the search box to find specific songs/albums/members
3. **Filter** - Click "Group", "Solo", or "Unit" to filter
4. **Play** - Click any song to play it instantly
5. **Navigate** - Use ⏮️ ▶️ ⏭️ 🔀 controls
6. **Enjoy** - Immerse yourself in BTS's music! 💜

### For Developers 💻
```bash
# Clone the repository
git clone https://github.com/shafaq0410/BTS_Discography.git

# Navigate to folder
cd BTS_Discography

# Open in browser
# Just open index.html in any modern browser!

# Or serve locally
npx serve
```

## 📝 Contributing Guide

We welcome all ARMY contributions! Here's the format for adding songs:
```javascript
{
  "title": "Song Title",
  "artist": "BTS / Member Name",
  "album": "YEAR — Album Name",
  "year": 2023,
  "type": "group", // or "solo" or "unit"
  "members": "Member Names", // for solo/unit tracks
  "youtubeId": "VIDEO_ID_HERE", // from youtube.com/watch?v=VIDEO_ID_HERE
  "image": "IMAGE_URL_HERE"
}
```

**Steps to Contribute:**
1. Fork this repository
2. Create a new branch (`git checkout -b add-new-songs`)
3. Make your changes to `index.html`
4. Commit (`git commit -m 'Added [Song Name]'`)
5. Push (`git push origin add-new-songs`)
6. Open a Pull Request 💜
